showStartTrackingDialog function

Future<Map<String, dynamic>> showStartTrackingDialog()

Opens a dialog to ask the user if tracking should be started Returns a Future that contains a Map when the dialog is closed The Map contains a key "startTracking" which value indicates whether or not tracking should be started If "startTracking" is true the Map also contains a key "label" which value is either "indoor" or "outdoor" depending on what the user says the tracking should be labeled with

Implementation

Future<Map<String, dynamic>> showStartTrackingDialog() => Get.dialog(StartTrackingDialog(), barrierDismissible: false);
