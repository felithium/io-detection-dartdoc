Future<List<Map<String, dynamic>>> readAll()

Reads all iod data from the db

Implementation

static Future<List<Map<String, dynamic>>> readAll() async {
  List<String> iodDateIds = List<String>.from(await StoreRef.main().record("iodDateIds").get(_db));
  return Future.wait(iodDateIds.map((iodDateId) async => await StoreRef.main().record(iodDateId).get(_db)));
}
