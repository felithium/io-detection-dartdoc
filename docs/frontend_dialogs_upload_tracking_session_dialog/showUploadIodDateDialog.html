showUploadIodDateDialog function

Future<Map<String, dynamic>> showUploadIodDateDialog()

Opens a dialog to make the user confirm whether or not the chosen date should be uploaded Returns a Future that contains a Map when the dialog is closed The Map contains a key "upload" which value indicates whether or not upload should be started If "upload" is true the Map also contains a key "deleteAfterUpload" which value indicates whether or not the user wants the iod date to be deleted after the upload

Implementation
